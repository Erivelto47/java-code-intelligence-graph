# Operational Decisions

Este diretorio registra decisoes operacionais do harness.

Exemplos:

- branch strategy;
- report policy;
- merge policy;
- artifact policy;
- convencoes de handoff;
- criterios de versionamento de reports.

Este diretorio nao e a mesma coisa que Decision Trace do produto. Nao confundir
com:

- `.code-atlas/decisions/`;
- `com.codeatlas.core.decision`;
- artefatos de Decision Trace;
- contratos ou fixtures de decisao gerados pelo Code Atlas.

Decisoes aqui explicam como o trabalho e conduzido. Elas nao representam
decisoes extraidas do codigo Java analisado.
